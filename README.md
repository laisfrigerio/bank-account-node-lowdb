# Bank Account 🏦

Bank Account Application with lowDb

## 🛠️ Stack

- NodeJS: v14
- Express: to control routes
- lowDb: json database
- Jest and supertest

## Run

```
    yarn install
    yarn dev
```

## Run Tests

```
    yarn test
    yarn test:coverage
```
